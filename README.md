# A/B Testing of the Marketing Campaign   
## Мета   
Дослідити вплив нової рекламної кампанії на продажі   

## Опис датасету   
Датасет містить дані A/B тесту рекламних кампаній   
Джерело: https://www.kaggle.com/datasets/faviovaz/marketing-ab-testing   
Колонки:  
- user id (унікальний код клієнта)  
- test group (ad - дослідна група, psa - контрольна група)  
- converted (факт конверсії)   
- total ads (загальна кількість показаної реклами)  
- most ads day (день тижня, в який найчастіше показано рекламу)  
- most ads hour (година, о котрій найчастіше показано рекламу)  
Розмір: 588,101 спостережень  

## Гіпотеза  
H₀ (нульова гіпотеза):    
Конверсія в групі ad не відрізняється від контрольної групи    
H₁ (альтернативна):  
Конверсія в групі ad відрізняється (або більша) від контрольної  

## Метод та результати  
Для перевірки гіпотези використано Z-test для пропорцій  

Конверсія ad = 2.55%  
Конверсія psa = 1.79%  
p-value = 0.000  

Різниця між групами статистично значуща  

## Висновок  
Група ad показала вищу ефективність      
Рекомендація: масштабувати рекламну кампанію
